# Pre-Game Strategy

## Options

### 1. Random Card
System generates full card.

---

### 2. Custom Card

Player can:
- select specific events
- assign preferred events
- system fills remaining slots

---

## Event Point Values

Each event has:
- difficulty
- rarity
- point_value

---

## Tradeoff

Higher point cards:
- harder to complete
- higher reward

---

## Goal

Give player agency without complexity.
