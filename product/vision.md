# Sports Watch Party Games

## Vision
Make watching sports more interactive, social, and competitive through simple, shareable games.

## Core Idea
Users create or join live games tied to real-world events (sports, watch parties). Players receive randomized or semi-custom cards and compete to complete event-based objectives.

## Core Loop

1. Create or join a game
2. Receive a card (random or customized)
3. Watch the game
4. Events are scored live
5. Progress updates in real time
6. First player to complete objective claims win
7. Results + scores shared
8. Play again

## Design Principles

- Frictionless joining (link-based)
- Mobile-first
- Casual-first, skill-layered
- Real-time shared experience
- Replayable sessions

## Game Modes (MVP)

- Quick Play (complete all events)
- Streak Mode (complete events in order)

## Skill Philosophy

- Skill occurs:
  - before game (card construction)
  - at win moment (claim timing)
- No heavy interaction during gameplay
