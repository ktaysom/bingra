# Roadmap

## Phase 1 — MVP (Local Live Games)

Goal: enable friend groups to play live bingo-style games via link

### Core Features
- Create game
- Join game via link (no account required)
- Generate card (random or semi-custom)
- Quick Play mode
- Streak Mode
- Manual scoring (host/scorer)
- Live updates via realtime
- First-to-claim winner detection
- Basic scoring system
- Rematch flow

---

## Phase 2 — Skill + Retention

- Event point values
- Custom card builder (pre-game strategy)
- "Call Your Shot" mechanic
- Career points system
- Score multipliers (player count, first-to-claim)
- Player identity persistence

---

## Phase 3 — Growth

- Public games
- Templates (NBA, NCAA, NFL, etc.)
- Shareable results cards
- Leaderboards
- Spectator mode

---

## Phase 4 — Advanced

- Auto-scoring via sports APIs
- Difficulty tiers
- Multi-round matches
- Event weighting system
- Creator templates marketplace
