Rules

❌ Don’t:

🧩 UI Design Principles
1. Card-Based Layout
Everything feels like a “game card”
2. Clear State Feedback
Green = success
Gray = inactive
Bold = actionable
3. Motion (future)
Subtle transitions
Real-time updates feel alive
